# Advanced Backend

This repository documents my journey toward becoming a backend engineer capable of building scalable, production-ready systems. Instead of focusing on a single project, it serves as a collection of experiments, notes, and hands-on implementations that help me understand how modern backend applications are designed, deployed, and maintained.

Throughout this repository, I'll explore topics such as Docker, Redis, system design, distributed systems, databases, caching, authentication, cloud technologies, and DevOps practices. My goal is not only to learn the theory behind these technologies but also to implement them through practical examples and small projects that demonstrate real-world use cases.

As artificial intelligence becomes an increasingly important part of software development, this repository will also include my exploration of AI engineering concepts such as LangChain, Vector Databases, Embeddings, Retrieval-Augmented Generation (RAG), AI agents, and building intelligent backend services powered by large language models.

This repository will grow over time as I continue learning, experimenting, and building. It is intended to be both a personal knowledge base and a record of my progress, where every new topic adds another building block toward understanding how large-scale, production systems are engineered.
